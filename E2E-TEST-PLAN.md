# E2E 테스트 계획 - OT-CHAWON

> **앱 개요**: AI 3D 가상 피팅 패션 쇼핑몰
> **URL**: http://localhost:3000
> **도구**: Playwright MCP
> **작성일**: 2026-04-02
> **최종 수정**: 2026-04-02 (Playwright 탐색 기반 추가/수정)

---

## 테스트 우선순위

```
P0 (Critical) : 회원가입 → 로그인 → 상품 탐색 → 장바구니 → 주문 결제
P1 (High)     : 검색, 카테고리/사이드 필터, 마이페이지 탭, AI 피팅 플로우
P2 (Medium)   : 좋아요, 비로그인 접근 제한, 에러 페이지, 3D 뷰어, 약관 동의
P3 (Low)      : 푸터 링크, 슬라이더, 피팅 히스토리, 뷰 토글, 관련 상품
```
